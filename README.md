# SPTB Integrated System V3.0

## Deployment

> [!CAUTION]
> **WARNING:** Anda **WAJIB** memasukkan `GEMINI_API_KEY` ke dalam ruangan 'Environment variables' di papan pemuka Netlify (Site settings > Environment variables) sebelum melakukan deployment. Jika tidak, proses build akan gagal kerana fail `vite.config.ts` memerlukannya.

## Setup

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`
